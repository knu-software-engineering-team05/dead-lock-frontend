# [소프트웨어공학] DeadLock

## About

> 해당 프로젝트는 소프트웨어 공학 수업에서 진행 중이며, `뇌졸중 위험 예방 및 관리 시스템`의 Frontend Repo 입니다.

---

## Usage

1. Git Clone

```cmd
$ git clone https://github.com/knu-software-engineering-team05/dead-lock-frontend
```

2. Install package (* yarn을 추천드립니다.)

```cmd
$ yarn

# or

$ npm install
```

3. Start

```cmd
$ yarn start

# or

$ npm run start
```

4. Build & Deploy (TBD)

```cmd
$ yarn build

# or

$ npm run build
```
